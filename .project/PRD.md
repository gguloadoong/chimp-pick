# 침팬지픽 PRD (Product Requirements Document)

> 최종 수정: 2026-03-20 | 작성: 김가즈아 (PM)

## 1. 제품 비전

**"누구나 30초 만에 주식/코인 등락을 예측하는 B급 밈 배틀 게임"**

### 문제 정의
- MZ세대는 주식/코인에 관심 많지만, 실제 투자는 부담스러움
- 기존 모의투자 앱은 재미없고 진지함
- 예측 게임 시장에 B급 감성의 킬러 앱 부재

### 솔루션
- 실제 시세 기반 UP/DOWN 바이너리 예측
- 가상 재화(바나나코인)로 부담 제로
- B급 밈 감성 + 게이미피케이션 = 중독성

### 타겟 유저
- **1차**: 20-30대 주식/코인 관심 MZ세대
- **2차**: 캐주얼 모바일 게이머
- **3차**: 금융 콘텐츠 소비자

## 2. 핵심 기능 (MVP)

### 2.1 예측 게임 (Core Loop)
- 종목 선택: KOSPI/KOSDAQ 주요 종목, BTC, ETH, 밈코인
- 타임프레임: 1분, 5분, 1시간, 1일
- 방향 선택: UP 🟢 / DOWN 🔴
- 베팅: 바나나코인 1~50BC
- 결과: 예측 적중 시 베팅액 × 1.8 회수, 실패 시 소멸
- 실시간 차트: 캔들차트 + 예측 구간 하이라이트

### 2.2 바나나코인 경제
- 초기 지급: 100BC (가입 보너스)
- 일일 출석: 10BC
- 연승 보너스: 5BC × 연승 수
- 친구 초대: 50BC
- 인앱 구매: ₩1,000 = 100BC (선택)

### 2.3 랭킹 시스템
- 일간/주간/월간/전체 랭킹
- 승률, 연승 기록, 총 수익률 기준
- 침팬지 진화 단계: 원숭이(1) → 침팬지(2) → 고릴라(3) → 킹콩(4) → 전설의침팬지(5)

### 2.4 소셜 기능
- 예측 결과 공유 카드 (이미지 자동 생성)
- 친구 대결 (1:1 같은 종목 동시 예측)
- 프로필: 전적, 칭호, 연승 기록

### 2.5 유저 인증
- 소셜 로그인: 카카오, 구글, 애플
- 닉네임 설정 (중복 불가)
- 게스트 모드: 로그인 없이 3회 체험 후 가입 유도

## 3. 비기능 요구사항

### 성능
- 예측 제출 → 서버 반영: < 500ms
- 시세 갱신 주기: 실시간 (WebSocket)
- 결과 판정 지연: < 3초
- 동시 접속: 10,000명 지원 (런칭 기준)

### 보안
- 실제 금전 거래 없음 (도박 규제 회피)
- 시세 조작 방지: 외부 API 기준가 사용
- 계정 도용 방지: JWT + Refresh Token

### 접근성
- 모바일 퍼스트 (PWA)
- iOS Safari, Android Chrome 지원
- 다크 모드 기본

## 4. 유저 스토리

### Epic 1: 첫 예측
```
AS A 신규 유저
I WANT TO 가입 후 바로 첫 예측을 하고 싶다
SO THAT 서비스의 재미를 즉시 체험할 수 있다

수용 기준:
- 가입 후 30초 이내 첫 예측 가능
- 종목 추천 (인기 종목 상위 3개)
- 튜토리얼 오버레이 (3단계)
- 첫 예측 무료 (바나나코인 차감 없음)
```

### Epic 2: 일일 루틴
```
AS A 기존 유저
I WANT TO 매일 접속해서 예측하는 습관을 만들고 싶다
SO THAT 랭킹도 올리고 바나나코인도 모을 수 있다

수용 기준:
- 일일 출석 보상 (10BC)
- 일일 미션 (예측 3회, 적중 1회 등)
- 스트릭 카운터 (연속 출석 일수)
- 푸시 알림 (설정 가능)
```

### Epic 3: 자랑하기
```
AS A 예측 적중 유저
I WANT TO 내 결과를 SNS에 공유하고 싶다
SO THAT 친구들에게 자랑하고 초대할 수 있다

수용 기준:
- 결과 공유 카드 이미지 자동 생성
- 카카오톡, 인스타, 트위터 공유
- 초대 링크 포함 (친구 가입 시 50BC 보상)
```

## 5. 제외 범위 (Out of Scope - MVP)

- 실제 금전 거래/베팅
- 해외 주식 (v2에서)
- 자동 예측 봇
- 채팅/커뮤니티 (v2에서)
- NFT/블록체인 연동
