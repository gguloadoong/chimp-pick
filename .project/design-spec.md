# 침팬지픽 디자인 스펙

> 최종 수정: 2026-03-20 | 작성: 박도파민 (Designer)

## 1. 디자인 원칙

1. **도파민 퍼스트**: 모든 인터랙션에 즉각적 시각 피드백
2. **엄지 영역 최적화**: 핵심 버튼은 하단 엄지 범위 내
3. **B급이되 조잡하지 않게**: 밈 감성이지만 UX는 프로페셔널
4. **다크 모드 기본**: 트레이딩 앱 감성 + 눈 피로 감소

## 4. 캐릭터 시스템

### 침팬지 진화 단계
| 단계 | 이름 | 조건 | 비주얼 |
|------|------|------|--------|
| 1 | 아기원숭이 | 가입 | 작고 귀여운 원숭이 |
| 2 | 침팬지 | 50승 | 바나나 들고 있는 침팬지 |
| 3 | 고릴라 | 200승 | 선글라스 쓴 고릴라 |
| 4 | 킹콩 | 500승 | 빌딩 위의 킹콩 |
| 5 | 전설의침팬지 | 1000승 | 금색 왕관 + 다이아몬드핸즈 |

### 감정 표현
- 승리: 선글라스 + 바나나 흔들기
- 패배: OTL 포즈 + 눈물
- 연승: 불꽃 이펙트 + 로켓
- 대기: 바나나 먹는 중

## 5. 애니메이션 가이드

### 마이크로 인터랙션
- **버튼 터치**: scale(0.95) → scale(1.0) + ripple (150ms)
- **예측 제출**: 버튼 → 로켓 발사 애니메이션 → 대기 상태
- **가격 변동**: 숫자 롤링 (odometer 스타일)
- **바나나코인 변동**: 카운트업/다운 + 흔들림

### 전환 애니메이션
- 페이지 전환: slide + fade (300ms, ease-out)
- 모달: scale(0.9→1.0) + fade (200ms)
- 결과 오버레이: 하단에서 슬라이드업 (400ms, spring)
