# 프로젝트 킥오프 미팅 🦍

## 진행 절차

### 1. 팀 소집
다음 에이전트를 순서대로 호출하여 킥오프 미팅을 진행합니다:

1. **김가즈아 (PM)**: 프로젝트 개요 발표
   - 침팬지픽 비전 및 목표 공유
   - MVP 스코프 확인
   - 스프린트 일정 공유

2. **박도파민 (Designer)**: 디자인 방향 발표
   - 디자인 시스템 개요
   - 핵심 화면 와이어프레임 공유
   - B급 밈 감성 톤앤매너 가이드

3. **이떡상 (FE) + 최풀매수 (BE)**: 기술 스택 발표
   - 아키텍처 개요
   - 기술 선택 이유
   - 개발 환경 세팅 계획

4. **한물타기 (Strategist)**: 그로스 전략 발표
   - 타겟 유저 정의
   - 핵심 메트릭
   - 런칭 전략

5. **장손절 (QA)**: 품질 전략 발표
   - 테스트 전략
   - 품질 게이트 기준
   - QA 일정

### 2. Q&A
- 각 에이전트의 질문/우려사항 수집
- 블로커 식별 및 해결 방안 논의

### 3. 액션 아이템
- 에이전트별 다음 스텝 확인
- 첫 스프린트 시작일 합의

### 4. 기록
- 회의록을 `.project/meeting-notes/kickoff-{날짜}.md`에 저장
