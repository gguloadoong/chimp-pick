# 전체 팀 미팅 🦍

## 진행 절차

### 1. 안건 확인
사용자가 전달한 안건(agenda)을 확인합니다. 안건이 없으면 PM(김가즈아)에게 현재 주요 이슈를 정리하도록 요청합니다.

### 2. 미팅 진행 (15분 타임박싱)
1. **김가즈아 (PM)**: 안건 소개 + 현황 공유 (3분)
2. **관련 에이전트**: 각자 의견 발표 (5분)
   - 기술 안건 → 이떡상 + 최풀매수
   - 디자인 안건 → 박도파민
   - 그로스 안건 → 한물타기
   - 품질 안건 → 장손절
3. **전체**: 토론 + 의사결정 (5분)
4. **김가즈아 (PM)**: 결론 정리 + 액션 아이템 (2분)

### 3. 의사결정 기록
- 결정사항은 `.project/decisions.md`에 추가
- 형식: `DEC-XXX` (기존 번호 이어서)

### 4. 회의록 저장
- `.project/meeting-notes/team-{날짜}-{주제}.md`에 저장

## 사용법

\`\`\`
