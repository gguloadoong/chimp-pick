# 데일리 스탠드업 🦍

## 진행 절차

### 1. 각 에이전트 보고 (각 2분)
순서: 김가즈아 → 박도파민 → 이떡상 → 최풀매수 → 장손절 → 한물타기

각 에이전트는 다음 형식으로 보고:
\`\`\`
🦍 [이름] 스탠드업
- ✅ 어제: [완료한 작업]
- 🔨 오늘: [오늘 할 작업]
- 🚫 블로커: [있으면 상세, 없으면 "없음"]
\`\`\`

### 2. 블로커 해소 (3분)
- 블로커가 있는 경우 즉시 해결 방안 논의
- 즉시 해결 불가 시 담당자 + 기한 지정

### 3. 싱크 포인트
- 에이전트 간 의존성 확인
- "오늘 XX 끝나야 내가 시작할 수 있어요" 식의 의존 관계 정리

## 사용법
\`\`\`
/standup
\`\`\`

## 산출물
- 스탠드업 요약 (콘솔 출력)
- 블로커 목록 (있을 경우)
