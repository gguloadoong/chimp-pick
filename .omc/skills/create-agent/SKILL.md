# 새 에이전트 생성 🦍+

## 개요
침팬지픽 팀에 새로운 전문 에이전트를 추가합니다.
침팬지픽 B급 감성에 맞는 닉네임과 페르소나를 부여합니다.

## 진행 절차

### 1. 역할 정의
사용자가 요청한 역할을 분석하여 다음을 결정:
- 역할명 (예: DevOps, Data Engineer, 마케터)
- B급 닉네임 (주식/코인 용어 기반)
- 배경 회사 (실존 기업 2곳)

### 2. 에이전트 파일 생성
`.claude/agents/{역할}.md` 파일을 9레이어 구조로 생성:

\`\`\`
Layer 1: 정체성 (Identity)
Layer 2: 핵심 역량 (Core Competencies)
Layer 3: 도메인 전문성 / 기술 스택
Layer 4: 커뮤니케이션 스타일
Layer 5: 의사결정 프레임워크 / 아키텍처
Layer 6: 핵심 산출물 / 프레임워크
Layer 7: 협업 프로토콜
Layer 8: 산출물 / 도구
Layer 9: 금기사항 (Anti-patterns)
\`\`\`

### 3. CLAUDE.md 업데이트
- 팀 구성 테이블에 새 에이전트 추가

### 4. 환영 인사
- 기존 팀원(김가즈아)이 새 멤버 환영

## 사용법
\`\`\`
/create-agent "DevOps 엔지니어"
/create-agent "데이터 엔지니어 — 실시간 분석 전문"
/create-agent "마케터 — SNS 바이럴 전문"
\`\`\`

## 닉네임 생성 규칙
- 주식/코인 용어 기반: 물타기, 떡상, 풀매수, 손절, 가즈아 등
- 성씨 + 용어 조합: 김/이/박/최/장/한/정/오/유/송 등
- 예: 정익절(익절 전문), 오존버(존버 전문), 유바닥(바닥매수 전문)
