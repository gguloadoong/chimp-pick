## 2. 유닛 테스트

### BE 핵심 테스트 대상
- `GameService.createPrediction()` — 예측 생성 로직
- `GameService.resolvePrediction()` — 결과 판정 로직
- `RewardService.processReward()` — 바나나코인 증감
- `PriceService.validatePrice()` — 시세 유효성 검증
- `RankingService.calculateRanking()` — 랭킹 집계

### FE 핵심 테스트 대상
- `usePrediction` 훅 — 예측 상태 관리
- `usePrice` 훅 — 실시간 가격 처리
- `PredictionPanel` — 입력 유효성 검증
- `ResultOverlay` — 결과별 분기 렌더링

## 4. E2E 테스트 (5대 핵심 플로우)

### E2E-001: 회원가입 → 첫 예측
```
1. 메인 페이지 접속
2. 카카오 로그인 (mock)
3. 닉네임 설정
4. 종목 선택 (BTC)
5. UP 예측 + 10BC 베팅
6. 예측 확인 화면 표시
확인: 바나나코인 90BC로 감소
```

### E2E-002: 예측 → 결과 → 보상
```
1. 예측 생성 (BTC UP 10BC)
2. 타이머 대기 (테스트용 1초)
3. 결과 판정 (mock: WIN)
4. 결과 화면 표시 (적중 애니메이션)
확인: 바나나코인 +18BC (10 × 1.8)
```

### E2E-003: 랭킹 확인
```
1. 랭킹 페이지 이동
2. 주간 탭 선택
3. 목록 로딩
4. 내 순위 확인
확인: 랭킹 데이터 표시 + 내 순위 하단 고정
```

### E2E-004: 프로필 확인
```
1. 프로필 페이지 이동
2. 전적 표시 확인
3. 종목별 성적 확인
확인: 승/패 수치 = 히스토리 합계
```

### E2E-005: 공유 카드 생성
```
1. 예측 결과 화면
2. 공유 버튼 클릭
3. 카드 이미지 생성
확인: 이미지에 결과/닉네임/전적 포함
```

## 6. 버그 트래킹

### 심각도 정의
- **P0**: 서비스 불가, 금액 오류 → 즉시 핫픽스
- **P1**: 핵심 기능 장애 → 당일 수정
- **P2**: 비핵심 이슈 → 다음 스프린트
- **P3**: 미세 이슈 → 백로그

### 릴리즈 기준
- P0 버그: 0건
- P1 버그: 0건
- E2E 5개 플로우: 전체 통과
- 유닛 커버리지: 80%+
