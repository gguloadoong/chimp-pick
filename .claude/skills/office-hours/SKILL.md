# 오피스 아워 💡

아이디어나 계획을 검증합니다. 팀 전원이 악마의 변호인 역할을 합니다.

## 6 Forcing Questions

1. **이 기능/아이디어가 없으면 사용자는 어떻게 해결하나요?**
   - 대안이 있으면 → 차별화 포인트를 다시 정의해야 합니다.

2. **왜 지금이 적기인가요?**
   - 시장, 기술, 규제 관점에서 지금이 맞는지 검토.

3. **첫 100명의 사용자를 어떻게 모을 건가요?**
   - 구체적인 채널과 메시지가 없으면 → 실행 불가.

4. **경쟁사가 따라하면 방어선은?**
   - 카카오/토스가 복사하면 어떻게 대응할 건지.

5. **성장 모델은?**
   - 바이럴, 광고, 파트너십 중 어느 것이 현실적인가.

6. **1년 후 성공의 모습은?**
   - 구체적인 수치 (MAU, 리텐션, 매출) 포함.

## 팀 역할

- **김가즈아 (PM)**: 사용자 관점 검증
- **한물타기 (Strategist)**: 시장/경쟁 관점
- **이떡상 (FE) + 최풀매수 (BE)**: 기술 실현 가능성
- **박도파민 (Designer)**: UX 관점
- **장손절 (QA)**: 엣지케이스, 실패 시나리오

## 결론

- 통과 → backlog에 추가 + 우선순위 배정
- 미통과 → 구체적인 약점 목록 + 수정 방향 제시
