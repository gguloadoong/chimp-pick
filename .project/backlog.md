# 침팬지픽 백로그

> 최종 수정: 2026-03-20 | 관리: 김가즈아 (PM)

## 우선순위 기준
- **P0**: 런칭 필수 (Must Have)
- **P1**: 런칭 시 있으면 좋음 (Should Have)
- **P2**: 런칭 후 (Could Have)
- **P3**: 나중에 (Won't Have Now)

## P0 — Must Have (MVP)

### 인증
- [ ] 카카오 소셜 로그인
- [ ] 게스트 모드 (3회 체험)
- [ ] JWT + Refresh Token
- [ ] 닉네임 설정 (중복 체크)

### 게임 코어
- [ ] 종목 선택 UI (코인 5개 + 주식 5개)
- [ ] 타임프레임 선택 (1분, 5분, 1시간, 1일)
- [ ] UP/DOWN 예측 버튼
- [ ] 바나나코인 베팅 (1~50BC)
- [ ] 실시간 캔들 차트
- [ ] 현재가 + 변동률 표시
- [ ] 예측 결과 판정 (서버)
- [ ] 결과 화면 + 애니메이션
- [ ] 바나나코인 증감 처리

### 시세 데이터
- [ ] Upbit 코인 시세 수집 (WebSocket)
- [ ] KIS 주식 시세 수집
- [ ] 캔들 데이터 집계 + 저장
- [ ] 실시간 브로드캐스트 (Socket.IO)

### 인프라
- [ ] 프로젝트 scaffolding
- [ ] DB 스키마 + 마이그레이션
- [ ] CI/CD 파이프라인
- [ ] 스테이징 환경
- [ ] 모니터링 (Sentry + 헬스체크)

- [ ] 결과 공유 카드
- [ ] 친구 초대 (50BC 보상)

### 게이미피케이션
- [ ] 침팬지 진화 시스템
- [ ] 칭호 시스템
- [ ] 연승 보너스
- [ ] 일일 출석 보상

### UX
- [ ] 온보딩 튜토리얼
- [ ] 푸시 알림
- [ ] 로딩/에러 상태 UI
