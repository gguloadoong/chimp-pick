## 1. 배틀패스 구조 설계

### 1-1. 시즌 기간

| 항목 | 설계 |
|------|------|
| 시즌 주기 | **4주 (28일)** |
| 시즌 번호 | Season 1, 2, 3... (연 13시즌) |
| 시즌 테마 | 매 시즌 밈 테마 변경 (예: S1 "우주원숭이", S2 "정글탈출", S3 "월스트리트침팬지") |
| 시즌 간 휴식 | 없음 — 종료 즉시 다음 시즌 시작 (FOMO 극대화) |
| 시즌 리셋 | 티어 진행도만 리셋. 획득한 보상은 영구 보유 |

**4주 근거:**
- 2주는 너무 짧아 무과금 유저가 최종 티어 도달 불가 → 좌절감
- 6주는 중반 이후 동기 저하 → 이탈
- 4주 = 주 5일 플레이 기준 최종 티어 도달 가능 (무과금) → 성취감 + 다음 시즌 재구매 유도

### 1-2. 티어 구조

**총 30티어** (무료 트랙 + 프리미엄 트랙 병렬)

| 구간 | 티어 | 필요 XP/티어 | 누적 XP | 설계 의도 |
|------|------|-------------|---------|-----------|
| 초반 | 1~10 | 100 XP | 1,000 | 빠른 보상 → 도파민 점화 |
| 중반 | 11~20 | 150 XP | 2,500 | 프리미엄 보상 맛보기 → 구매 전환 유도 |
| 후반 | 21~30 | 200 XP | 4,500 | 희귀 보상 집중 → 완주 동기 |

**총 필요 XP: 4,500 XP** (28일 기준 일 161 XP 필요)

### 1-3. XP 획득 경로

| 행동 | XP | 일일 상한 | 설계 의도 |
|------|-----|----------|-----------|
| 예측 참여 (1회) | 20 XP | 5회 = 100 XP | 코어 루프 활성화 |
| 예측 적중 | +10 XP (보너스) | 제한 없음 | 실력 보상 |
| 일일 로그인 | 30 XP | 1회 | 출석 습관 형성 |
| 3연승 달성 | 50 XP | 1회/일 | 연승 도전 동기 |
| 주간 미션 완료 | 100 XP | 주 3개 | 중장기 목표 |
| 친구 초대 (가입) | 200 XP | 제한 없음 | 바이럴 부스트 |
| 결과 공유 (SNS) | 15 XP | 3회 = 45 XP | K-factor 상승 |

**일일 달성 가능 XP (적극 플레이):** 약 200~250 XP
**일일 달성 가능 XP (캐주얼):** 약 80~120 XP

캐주얼 유저도 28일 중 25일 플레이 시 최종 티어 도달 가능 (25일 x 180 XP = 4,500 XP).

### 1-4. 무료 vs 프리미엄 트랙

```
티어 1  ────── 티어 10 ────── 티어 20 ────── 티어 30
  |                |               |               |
[무료]  무료 보상   무료 보상        무료 보상        무료 최종 보상
[유료]  잠금 🔒    프리미엄 보상    프리미엄 보상     전설급 최종 보상
```

- **무료 트랙**: 모든 유저 접근 가능. FREE 코인, 기본 칭호, 일반 이펙트
- **프리미엄 트랙**: 배틀패스 구매 유저만. 프리미엄 스킨, 희귀 칭호, 특수 이펙트, PAID 코인

### 2-3. 보상 설계 원칙

1. **PAID 코인 환급률 설계**: 프리미엄 트랙의 PAID 코인 총합(430 BC)이 배틀패스 가격(약 500~600 BC 상당)의 70~85% 수준 → "다음 시즌도 사면 거의 본전" 인식 유도
2. **코스메틱 Only**: 모든 유료 보상은 외형/칭호/이펙트. 예측 승률이나 FREE 코인 획득에 영향 없음 (Pay-to-Win 금지)
3. **시즌 한정 FOMO**: 티어 10, 20, 30의 시즌 한정 아이템은 해당 시즌에서만 획득 가능 → 재구매 동기
4. **무료 트랙도 의미 있게**: 무료 유저에게도 190 BC + 칭호/이펙트 제공 → 게임 이탈 방지

## 3. 가격 전략

### 3-1. 배틀패스 가격

| 상품 | 가격 (KRW) | PAID 코인 환산 | 포지셔닝 |
|------|-----------|---------------|----------|
| **프리미엄 패스** | **4,900원** | 490 BC | 커피 한 잔 가격 — 심리적 저항선 최소화 |
| **프리미엄 패스 + 10티어 스킵** | **8,900원** | 890 BC | 바쁜 유저용 — 시즌 중반 구매자 대응 |
| **프리미엄 패스 + 20티어 스킵** | **12,900원** | 1,290 BC | 막판 구매자 — 시즌 종료 직전 FOMO 타겟 |

### 3-2. 가격 설계 근거

**4,900원 심리적 가격 분석:**
- 5,000원 미만 = "만 원도 안 해" 심리. 소액결제 저항 최소
- 스타벅스 아메리카노(4,500원)와 비슷 → "커피 대신 배틀패스" 프레이밍
- 국내 모바일 게임 배틀패스 평균: 5,000~11,000원 (배그모바일 10,000원, 브롤스타즈 8,900원)
- 침팬지픽은 캐주얼 타겟 → 하한선 공략이 전환율 극대화

**티어 스킵 번들 근거:**
- 시즌 중반(2주차) 신규 가입자 or 복귀 유저 → 남은 기간에 30티어 달성 불안 → 스킵 번들 구매
- 10티어 스킵 = 4,000원 추가 (티어당 400원) → "합리적" 인식
- 20티어 스킵 = 8,000원 추가 (티어당 400원) → 동일 단가 유지로 일관성

## 4. 앱스토어 / GRAC 심사 대응 포인트

### 4-1. Apple App Store (Guideline 5.3 도박)

| 심사 우려 | 대응 논리 |
|-----------|-----------|
| "배틀패스가 도박 요소 아닌가?" | 배틀패스는 확정 보상 시스템. 랜덤 요소 없음 (가챠/루트박스 아님). 티어별 보상이 사전 공개되어 있고, XP 누적으로 확정적으로 획득. |
| "PAID 코인이 예측에 사용되면?" | DEC-009 기준, PAID 코인은 코스메틱 전용. 서버 레벨에서 `coinType=PAID`는 `PREDICTION_SPEND` 차단. 코드로 증명 가능. |
| "배틀패스 보상으로 FREE 코인 지급하면?" | 무료 트랙의 FREE 코인은 무료 유저에게도 동일 지급. 배틀패스 구매 여부와 무관하게 무료 트랙은 열려 있음. 프리미엄 트랙은 PAID 코인만 지급. |
| "현금 전환 가능한가?" | 바나나코인은 어떤 경로로도 현금 전환 불가. 출금 기능 없음. 서비스 내 코스메틱 소비만 가능. |

**핵심 방어 논리:** "배틀패스 = 시즌 구독 상품. 확정 보상 + 코스메틱 전용 + 현금 전환 불가 = 도박 아님."

### 4-2. Apple Guideline 3.1.1 (인앱 구매)

| 요구사항 | 준수 방안 |
|----------|-----------|
| 디지털 콘텐츠는 IAP 필수 | 배틀패스, 코인 팩 모두 Apple IAP 사용 |
| 외부 결제 유도 금지 | 앱 내 결제만 안내. 외부 링크/QR 없음 |
| 구독 자동갱신 주의 | 배틀패스는 일회성 구매 (auto-renewable subscription 아님) → 자동갱신 관련 조항 해당 없음 |

### 4-3. Google Play 정책

| 요구사항 | 준수 방안 |
|----------|-----------|
| 도박 정책 | Apple과 동일 논리. 확정 보상 + 코스메틱 + 현금 전환 불가 |
| Real-money gambling 금지 | 바나나코인 = 게임 내 가상 재화. 현금 가치 없음 |
| Simulated gambling | 시뮬레이션 등급 표시 + 연령 제한 설정 |

### 4-4. GRAC (게임물관리위원회)

| 항목 | 대응 |
|------|------|
| 등급 분류 | **전체 이용가** 또는 **12세 이용가** 목표 |
| 사행성 판단 기준 | (1) 결과의 우연성 → 예측 게임이나 현금 보상 없음 (2) 사행심 유발 → 가상 재화만, 현금 전환 불가 (3) 과금 구조 → 확정 보상(배틀패스), 가챠 없음 |
