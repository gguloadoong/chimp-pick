### DEC-007: Price 카테고리 최소 타임프레임 5분으로 상향 (2026-03-25)
- **결정**: 30초 price 예측 제거. 최소 5분, 코어 타임프레임 1시간
- **대안**: 30초 유지, 1분 최소
- **이유**:
  - 30초 시세 변동 = 시장 노이즈. 어떤 분석도 edge 없음 → 슬롯머신 인식
  - 5분 이상이어야 "예측 근거" 형성 가능
  - 1시간 타임프레임 = 결과 대기 → 재방문 훅 → D1 리텐션 35~45% 기대
  - "오늘 예측 → 1시간 후 결과 확인 → 재방문" 습관 루프 설계
- **결정자**: CEO 확정 (2026-03-25)
- **상태**: 확정
